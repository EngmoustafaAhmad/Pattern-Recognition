# Pattern Recognition Repository

This repository contains explained lectures, notes, and code examples for Pattern Recognition.

## Contents
- `lectures/`: Lecture explanations, slides, and examples.
- `code/`: Reusable Python/Matlab code for various pattern recognition techniques.
- `datasets/`: Sample datasets for experimentation.
- `references/`: Useful papers and books on Pattern Recognition.

## Suggested Usage
1. Start with `lectures/lec01-introduction`.
2. Follow the lectures in order.
3. Explore the code in `code/` for hands-on practice.
